## 二、实现流程

### Step 1 - 搜索原版游戏
- 搜索《篝火：被遗忘的土地》(The Bonfire: Forsaken Lands)
- 分析核心玩法：昼夜循环、资源管理、工人系统、建造系统、怪物袭击、探索系统
- 确定技术栈：HTML5 Canvas + JavaScript (网页版先行)

### Step 2 - 搭建基础框架
1. **HTML结构**：Canvas画布 + 顶部信息栏 + 资源栏 + 底部操作栏 + 面板容器
2. **CSS样式**：深蓝/暗红国风配色，像素风格字体
3. **游戏循环**：`requestAnimationFrame` + 固定时间步长更新

### Step 3 - 核心系统实现
1. **昼夜循环**：时间推进 + 天空渐变 + 太阳/月亮 + 星星
2. **资源系统**：木材/食物采集、消耗、产出
3. **工人系统**：招募、分配工种、AI行为状态机
4. **建造系统**：多类型建筑 + 解锁条件
5. **战斗系统**：怪物生成 + 守卫追击 + 伤害计算

### Step 4 - 增强体验
1. **国风美术**：独立人物设计、东方神话怪物、水墨远山、国风松树
2. **动画系统**：行走腿部摆动、工具挥动、树木摇晃、攻击特效
3. **音效系统**：五声音阶古琴旋律(白天)、低沉恐怖氛围(夜晚)
4. **剧情系统**：开场序章 + 8个主线任务指引
5. **存档系统**：localStorage本地存档

## 四、核心代码模块

### 4.1 游戏循环
```javascript
function gameLoop(timestamp) {
    const deltaTime = (timestamp - lastTime) / 1000;
    accumulator += deltaTime;
    while (accumulator >= FIXED_TIME_STEP) {
        update(FIXED_TIME_STEP);  // 逻辑更新
        accumulator -= FIXED_TIME_STEP;
    }
    render();  // 画面渲染
    requestAnimationFrame(gameLoop);
}
```

### 4.2 昼夜切换
```javascript
Game.isDay = Game.time >= 6 && Game.time < 18;
if (wasDay && !Game.isDay) { onNightStart(); }  // 怪物生成
if (!wasDay && Game.isDay) { onDayStart(); }    // 天数+1
```

    // 11. 粒子效果

## 五、关键技术点

| 技术 | 实现方式 |
|---|---|
| 像素风格 | `image-rendering: pixelated` + Canvas绘制矩形/像素 |
| 国风配色 | 中国传统色彩：靛蓝、朱红、赭石、松绿、金黄 |
| 程序音效 | Web Audio API 振荡器 + 增益节点 + 滤波器 |
| 古琴音乐 | 五声音阶 (C D E G A) + 正弦波 + 颤音包络 |
| 动画系统 | `Date.now()` 帧计数器 + `Math.sin()` 周期性动画 |
| 屏幕震动 | Canvas `translate()` + 随机偏移 + 指数衰减 |
| 粒子系统 | 数组存储粒子对象，每帧更新位置/速度/生命周期 |
| 存档系统 | `localStorage.setItem('bonfire_save', JSON.stringify(data))` |
| 建筑解锁 | 配置对象定义解锁条件，UI动态检查显示 |
| 任务指引 | 任务数组 + 条件检查函数，完成后自动切换 |
